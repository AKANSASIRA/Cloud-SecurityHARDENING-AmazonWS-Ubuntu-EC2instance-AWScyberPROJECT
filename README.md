# AWS Cloud Security Hardening Project

## Overview
This project demonstrates deploying, configuring, and securing a cloud server on Amazon Web Services (AWS) using an Ubuntu EC2 instance. The goal is to gain hands-on experience in cloud security, firewall management, and basic vulnerability scanning.

## Tools Used
- AWS EC2 (Elastic Compute Cloud)
- Ubuntu Server (Linux)
- UFW (Uncomplicated Firewall)
- Nmap (Network scanner)

## Steps Performed
1. Deployed an EC2 instance using Ubuntu Server (Free Tier).  
2. Configured firewall rules to allow only essential ports (SSH: 22, HTTP: 80).  
3. Updated and upgraded the system for security patches.  
4. Installed Nmap and performed a local vulnerability scan.  
5. Captured screenshots of terminal outputs and EC2 dashboard for documentation.

## Results
- Firewall successfully enabled and only necessary ports open.  
- Nmap scan confirmed minimal exposure of services.  
- Project demonstrates basic cloud security practices and hands-on experience.
